# 🎨 Sistema de Autenticação - Frontend
Interface para interação com a API de autenticação.

## Objetivo
Permitir cadastro, login e recuperação de senha integrada ao backend.

## Tecnologias
- HTML, CSS, JavaScript  

## Funcionalidades
- Cadastro de usuário  
- Login  
- Solicitação de recuperação de senha  
- Integração com a API do backend

## Fluxo da Aplicação
1. Usuário se cadastra  
2. Usuário faz login  
3. Caso esqueça a senha, solicita recuperação  
4. Recebe e-mail de recuperação
